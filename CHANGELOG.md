# Changelog

All notable changes to the Faith Immaculate Academy CBT System will be documented in this file.

## [5.7.7] - 2026-06-05

### Added
- **Multi-Subject Exam Sidebar UI**: Introduced a sidebar navigation grid in `client/src/pages/exam-session.tsx` for multi-subject CBT tests. Shows section completion stats (e.g. `12/20 completed`), checkmarks, and dynamic navigation filter mapping.
- **Vercel ESM Serverless Builder**: Created `build-backend.js` utilizing `esbuild` to compile `api-src/[...route].ts` to `api/[...route].js`. Resolves runtime relative path ESM import bugs on Vercel deployment environments.
- **Missing Imports**: Resolved missing `BookOpen` and `useMemo` imports inside `exam-session.tsx` to ensure clean frontend compiles.

## [5.6.0] - 2026-06-02

### Added
- **AI-Powered Question Importer**: Created Gemini API-driven question extractor to parse unstructured Word (`.docx`) and notepad (`.txt`) documents using structural JSON schemas.
- **Structured Upload Preview Grid**: Added client-side visual preview matching questions before database updates.
- **Mammoth Integration**: Installed `mammoth` on the server for raw document text mining.

## [5.5.0] - 2026-05-30

### Added
- **Advanced Cohort Psychometrics**: Added class-wide and department-wide average analytics leaderboards.
- **Comparative Mastery Radar Charts**: Integrated Recharts Radar visualizations contrasting individual performance with cohort averages.
- **Academic Trajectory Area Charts**: Added historical growth tracking graphics.
- **A4 Printable Study Guides**: Created PDF-ready print action cards showing AI intervention paths.

### Optimized
- **Instant Analytics Load**: Rewrote calculations in `admin-analytics.tsx` from $O(N^3)$ to $O(N)$ hash-indexing.

## [5.1.0] - 2026-05-28

### Added
- **Interactive Question Bank Linker**: Implemented dynamic inline explorer in `admin-exam-details.tsx` to search, preview, and toggle links in questions based on exam parameters.
- **Multi-Variable Filters**: Added Term, Class, Department, and Subject filtering to the exams list.
- **Client-Side Pagination**: Implemented 10-records list limitation and navigation boundaries.

## [5.0.0] - 2026-05-26

### Added
- **Firestore Offline Sync**: Initialized Firestore Database integration with client-side IndexedDB caching, enabling offline resilience, real-time syncing, and multi-tab synchronization.
